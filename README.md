# GitHub Commercialization Analyzer

Этот проект автоматически анализирует 5 самых популярных репозиториев на GitHub и оценивает их потенциал коммерциализации с помощью API Google Gemini.

## Возможности
- Автоматический сбор данных о трендовых проектах
- Анализ с использованием Google Gemini API
- Автоматический запуск через GitHub Actions каждые 6 часов

## Установка
1. Склонируйте репозиторий
2. Установите зависимости: `pip install -r requirements.txt`
3. Добавьте ваш ключ API Gemini в `.env`
4. Запустите скрипт: `python main.py`
